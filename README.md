## 주요 프로젝트 및 성과

### [Knowledge Garden](https://github.com/ssyy3034/my-knowledge-garden)

> **자율적 학습 대시보드 및 지식 그래프 시각화**
> Next.js와 D3.js를 연동하여 Obsidian 데이터를 시각화하는 엔진을 구축했습니다. 단순한 기록을 넘어 성장을 데이터로 추적하고 아카이빙하는 엔지니어링 도구입니다.

> NoSQL(MongoDB)과 RDB의 상이한 데이터 스키마를 정규화하여 프론트엔드에서 일관되게 관리하는 어댑터 레이어를 설계했습니다. 대규모 그래프 시각화와 에디터 간의 실시간 동기화 성능 병목을 해결한 경험이 있습니다.

## 전문성 구현 사례

- **운영체제 설계**: Pintos Kernel 프로젝트를 통해 CPU 스케줄링 및 가상 메모리 관리 등 저수준 시스템의 동작 원리를 직접 구현하며 문제 해결 능력을 다졌습니다.
- **데이터 일관성 관리**: 프로젝트 성격에 따른 상이한 데이터 구조를 통합하여 프론트엔드 SSOT(Single Source of Truth)를 구축하고 유지하는 아키텍처링을 지향합니다.
- **렌더링 퍼포먼스**: 브라우저의 Critical Rendering Path에 대한 이해를 바탕으로, 대규모 인터랙티브 그래프 환경에서도 사용자 경험을 저해하지 않는 성능 최적화를 수행합니다.
